Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b> (tag não semântica b).

Nesta frase, tenho um termo em destaque usando a tag STRONG (tag semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (tag não semântica i).

Já nesta frase, temos um termo em ênfase usando a tag <em> (tag semântica em).

Texto marcado

Podemos criar também um texto marcado usando a tag MARK. (tag semântica mark)

E no outro parágrafos, temos outro texto marcado no final.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo. ( tag obsoleta big e tag semântica small)

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. (tag semântica del)

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. (tag semântica ins)

Podemos marcar um texto da mesma forma com a tag u. (Não semântica.)

Texto sobrescrito

Para inserir coisas do tipo x20+3. (tag sup)

Texto subscrito

Para inserir coisas do tipo H2O. (tag sub)
